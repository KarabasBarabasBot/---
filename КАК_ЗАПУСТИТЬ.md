# Крестики-нолики // Хакерский стиль

## Как запустить

Открой Терминал, вставь команду и нажми Enter:

```
cd "/Users/vladimir/VS код" && python3 -m http.server 8080
```

Затем открой браузер и перейди по адресу:

```
http://localhost:8080
```

Чтобы остановить сервер — нажми Ctrl+C в Терминале.

## Как работает игра

- **1-я, 3-я, 5-я партия** — ИИ играет на победу (невозможно выиграть)
- **2-я, 4-я, 6-я партия** — ИИ специально проигрывает
- Кнопка 🔊 в правом нижнем углу — выключить/включить музыку
